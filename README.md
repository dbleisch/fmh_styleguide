# fmh-frontend
FMH HTML-Templates

## Installation

```sh
git clone git@github.com:dbleisch/fmh-frontend.git
cd fmh-frontend
npm i
npm run build:dev
```
# mit webserver laufen lassen
npm run build:dev

# compilieren
npm run build:p

## Projekt bauen ohne Webpack-Dev-Server

```npm run build```

## Projekt bauen mit Webpack-Dev-Server

```npm run dev:build```
